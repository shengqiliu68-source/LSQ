# LSQ Website

企业产品周边定制与全球物流仓储业务网站。

## 常用命令

```bash
npm run dev
npm run lint
npm run typecheck
npm run build
```

本地预览默认地址：

```text
http://localhost:5173/
```

## 项目技术

- React
- Vite
- TypeScript
- Tailwind CSS
- ESLint
- Prettier

## 当前页面

- 全屏视频背景首页 Hero
- 周边定制能力模块
- 定制案例大卡片模块
- 业务优势模块
- 全球物流与仓储优势模块
- 整屏联系方式收尾页
