# LSQ Website

这是你的网页开发项目。环境已经配置为使用 VS Code + GitHub + Codex 开发。

## 常用命令

```bash
npm run dev
```

打开本地页面：

```text
http://localhost:3000
```

## 项目技术

- Next.js
- TypeScript
- Tailwind CSS
- ESLint
- Prettier

## AI 开发方式

以后你只需要告诉 Codex 你要做什么网页或修改什么内容，Codex 会按 `AGENTS.md` 中的规则自动开发、测试、提交和同步。

## 检查项目

```bash
npm run check
```
